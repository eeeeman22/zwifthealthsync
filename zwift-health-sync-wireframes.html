Design Notes & Key Features

  • Core Flow: User logs in with Zwift → authorizes Apple Health → sees recent rides → can manually sync or enable auto-sync
  • Dashboard Focus: Quick status check + recent rides list with key metrics visible at a glance
  • Ride Detail: Full metadata display including power, HR, cadence, elevation, gear, route map screenshot
  • Sync Strategy: Manual "Sync Now" button + optional auto-sync toggle for set-it-and-forget-it users
  • Data Priority: Power (watts), HR from external sensor, cadence, elevation, timing, distance, gear/bike selection
  • Minimal Premium: Clean UI with no bloat - just Zwift → Apple Health, nothing else
  • Visual Touch: Route map screenshot integration for each ride (makes it feel less clinical)
  • Settings Simplicity: Toggle what syncs, manage connections, that's it
